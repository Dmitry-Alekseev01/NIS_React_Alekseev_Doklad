## Запуск

```bash
ng serve
```

Приложение будет доступно по адресу `http://localhost:4200`.  
При необходимости порт можно сменить: `ng serve --port 4300`.

## Вход
Использовать можно любые данные (валидация только на заполнение полей):

- **Email:** `test@test.com`
- **Пароль:** `123456`

## Функции

- **Дашборд** – сводка по проектам и график активности.
- **Проекты** – список, детальная информация, создание/редактирование.
- **Задачи** – список, детали, форма создания.
- **Аналитика** – график сгорания задач и граф зависимостей.
- **Переключение языка** – русский / английский.


## Использованные технологии

| Технология           | Где использована                           |
|----------------------|--------------------------------------------|
| Angular 17           | Современный фреймворк                      |
| NgRx                 | Управление состоянием (проекты, задачи)    |
| ngx‑translate        | Интернационализация (переключение языков)  |
| Angular Material     | UI‑компоненты                              |
| ECharts              | Графики на дашборде и аналитике            |
| D3.js                | Кастомный граф зависимостей                |
| Socket.IO‑client     | Имитация real‑time                         |
| date‑fns             | Работа с датами                            |
| SCSS + BEM           | Стилизация                                 |
